getConsumptionData

  • getConsumptionData(supply, startDate?, endDate?, measurementType?, authorizedNif?): Promise<ConsumptionData[]>

  • Gets consumption data for a Supply in a given time range. Right now, Datadis only support queries for a natural month, dates provided would be truncated. Datadis takes both start and end dates as inclusive, this is, if you want to get consumption data from August, you must pass 2024/08/XX and 2024/08/XX. If you pass 2024/08/XX and 2024/09/XX you would be asking for all data for August and Sept.

    WARNING: This call can only be done once a day for the same time range.

    Parameters

    • supply: {
          cups: string;
          distributorCode: string;
          pointType: number;
      }

      a Supply.

      • cups: string

        a valid CUPS from a Supply.

      • distributorCode: string

        a valid distributor code from a Supply. Valid values: (1: Viesgo, 2: E-distribución, 3: E-redes, 4: ASEME, 5: UFD, 6: EOSA, 7:CIDE, 8: IDE).

      • pointType: number

        a valid point type from a Supply.

    • startDate: Date = ...

      Start date from the desired time range.

    • endDate: Date = ...

      End date from the desired time range.

    • measurementType: 0 | 1 = 0

      Set to 0 to get consumption data every hour. Set to 1 to get consumption data every 15 minutes (at the moment, this last case is only available for a combination of point type 1 or 2 and distributor code 2).

    • OptionalauthorizedNif: string

      the NIF of another person that has been authorized.

    Returns Promise<ConsumptionData[]>

    an array of ConsumptionData.

    Throws

    a DatadisAPIError if API call fails.

    Throws

    a DatadisError if something else fails.
